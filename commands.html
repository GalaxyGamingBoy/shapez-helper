# Command Name Action Format Example
1 encodeB64 Encodes file to Base64 encodeB64 [fileLocation] shapez-helper.exe encodeB64 .\Hello.txt
2 encodeB64Line Encodes text to Base64 encodeB64Line [text] shapez-helper.exe encodeB64Line "Hello, World!"
3 decodeB64 Decodes file from Base64 to a regular file (Extension + Name specified in settings) decodeB64 [filePath] shapez-helper.exe decodeB64 .\BEncodedData.txt
4 mergeFiles Merge all files with the selected extension in the /src/ folder. !! NOTE !! End Imports with this comment // IMPORT-END mergeFiles [fileExtension] [fileOutputName] shapez-helper.exe mergeFiles "js" "mergeMODFiles"
